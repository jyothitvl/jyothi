CSS Introduction

CSS stands for Cascading Style Sheets. CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

CSS Syntax

A CSS rule consists of a selector and a declaration block.

  • The selector points to the HTML element you want to style
  • The declaration block contains one or more declarations separated by semicolon
  • Each declaration includes a CSS propery and value, separated by colon
  • Multiple declarations are separated by semicolons, declaration blocks are surrounded by curly braces

We can divide CSS selectors into five categories:

CSS Selectors

We can divide CSS selectors into five categories:

  • Simple selectors: select elements based on name, id, class
  • Combinator selector: select elements based on a specific relationship between them
  • Pseudo-class selector: select element based on a certain state
  • Pseudo-elements selectors: select and style a part of an element
  • Attribute selectors: select elements based on an attribute or attribute value

CSS Combinators Selectors

A combinator is somthing explains the relationship between the selector.

A CSS selector contain more than one simple selector. Between the simple selectors, we can include a combinator.

There are four diffenet combinators in CSS:

Descendant selector (space):
-It matches all elements that are descendant of a specified element
Child selector (>)
- It selects all element that are the childern of a specified element
Adjacent sibling selector (+):
- It selects an element that is directly after another specific element. Sibling element must have the same parent element, and "adjacent" means immediately following.
General sibling selector (~):
- It selects all elements that are siblings of a specified element.

CSS Colors

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

CSS color Names:

By using predefined color name: Tomato, Orange, DodgerBlue, MediumSeaGreen, Gray, SlateBlue, Violet, LightGray.

CSS RGB Colors:

An RGB color represents RED, GREEN, and BLUE light sources.By using this formula rgb(red, green, blue).

Each parameter defines the intensity of the color between 0 and 255.

rgb(255,0,0)
-- is displayed as red.
rgb(0,0,0)
-- is displayed as black.
rgb(255,255,255)
-- is displayed as white.

CSS RGBA Colors:

It is an extension of RGB color values with an alpha channel which is specifies the opacity for a color.

The alpha parameter is a number between 0.0(fully transparent) and 1.0(not transparent at all)

rgb(255,0,0,0.5)
-- is displayed as 50% transparent.

CSS HEX colors:

A hexadecimal color is specified with: #RRGGBB, where the RR(red), GG(green) and BB(blue).A hexadecimal values between 00 and ff(same as decimal 0-255)

#ff0000
-- is displayed as red.
#000000
-- is displayed as black.
#ffffff
-- is displayed as white.

CSS HSL colors:

HSL stand for hue, saturation, and lightness.

hsl(hue, saturation, lightness)

  • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 green, 240 blue.
  • Saturation is a percentage vlue, 0% means a shade of gray, and 100% is the full color.
  • lightness is also percentage, 0% is black, 50% is neither light or dark, 100% is white.
hsl(0,100%,50%)
-- is displayed as red.
hsl(0,100%,0%)
-- is displayed as black.
hsl(0,100%,100%)
-- is displayed as white.

CSS HSLA colors:

It is an extension of HSL color values with an alpha channel which is specifies the opacity for a color.

The alpha parameter is a number between 0.0(fully transparent) and 1.0(not transparent at all)

CSS Backgrounds

The CSS background properties are used to add background effects for elements.CSS background properties:

  • Background-color
  • Background-image
  • Background-repeat
  • Background-attachment
  • Background-position
  • Background (shorthand property)

CSS background-color:

It specifies the background color of an element.

Opacity/Transparency

The Opacity property specifies the opacity of an element. It can take a value from 0.0-1.0. The lower value, the more transparent.

CSS Background-image:

It specifies an image to use as the background of an element. By default, The image is repeated.

CSS Background-repeat:

By default,it repeats an image both horizontally and vertically.

background-repeat: repeat-x;
--The image repeated only horizontally.
background-repeat: repeat-y;
--The image repeared only vertically
background-repeat: no-repeat;
--The image showing only once.
background-position: right top;
It specify the position of the image.

CSS Background-attachment:

It specifies whether the background image should scroll or be fixed

background-attachment:fixed;
--specify the image should be fixed.
background-attachment:scroll;
--specify the image should scroll with the rest of the page.

CSS Background Shorthand

Use the shorthand property to set the background properties in one declaration. When using the shorthand property the order of the property value is:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

CSS Borders

The CSS border properties allow you to specify the style, width, color of an element's border.

The border-style property specifies the type of border to display.

  • dotted - Defines a dotted border.
  • dashed - Defines a dashed border.
  • solid - Defines a solid border.
  • double - Defines a double border.
  • groove - Defines a 3D grooved border. the effect depends on the border-color value.
  • ridge - Defines a 3D ridge border. the effect depends on the border-color value.
  • inset - Defines a 3D inset border. the effect depends on the border-color value.
  • outset - Defines a 3D outset border. the effect depends on the border-color value.
  • none - Defines no border.
  • hidden - Defines a hidden border.

border-style property have four values (for the top border, right border, bottom border, and the left border)

If the border-style property has four values:

  • border-style: dotted solid double dashed;
    • top border is dotted
    • right border is solid
    • bottom border is double
    • left border is dashed

If the border-style property has three values:

  • border-style: dotted solid double;
    • top border is dotted
    • right and left border are solid
    • bottom border is double

If the border-style property has two values:

  • border-style: dotted solid;
    • top and bottom border are dotted
    • right and left border are solid

If the border-style property has one value:

  • border-style: dotted;
    • all four borders are dotted.

CSS Border Width

The border-width property specifies the width of the four borders. The width can set as a specific size(in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick.

CSS Border Color

The border-color property is used to set the color of the four borders. The color can be set by:

  • name-specify a color name, like "red"
  • HEX-specify a HEX value, like "#ff0000"
  • HEX-specify a HEX value, like "#ff0000"
  • HSL-specify a HSL value, like "hsl(0,100%,50%)"
  • transparent

CSS Margins

It is used to create space around the element, outside of any defined borders. it has property for specifying the margin for each side of an element same as border-style property:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

All the margin properties can have the following values:

  • margin--shorthand property--to specify all the margin properties
  • auto--to horizontally center the element within its container. The element should specify the width.
  • length-- specifies a margin in px, pt, cm, etc.
  • %-- specifies a margin in % of the width of the containing element.
  • inherit--specifies that the margin should be inherited from the parent element

Margin Collapse--Only top ans bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins.

CSS Padding

It is used to create space around the element's content, inside of any defined borders.CSS has property for specifying the padding for each side of an element same as border-style property:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

All the margin properties can have the following values:

  • padding--shorthand property--to specify all the padding properties
  • length-- specifies a margin in px, pt, cm, etc.
  • %-- specifies a margin in % of the width of the containing element.
  • inherit--specifies that the margin should be inherited from the parent element

CSS Height and Width

These properties are used to set the height and width of an element.Its do not include padding, borders, or margin.These properties may have the following values:

  • auto--This is default. The browser calculates the height and width.
  • length--Defines the height/width in px,cm etc.
  • %--Defines the height/width in percent of the containing block.
  • initial--Set the height and width to its default value.
  • Inherit--The height/width will be inherited from its parent value.

max-width:property is used to set the maximum width of an element. It can be specified in length values, like px,cm, etc. or in percent (%)
