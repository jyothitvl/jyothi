HTML Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more. style attribute syntax:
<tagname style="property:value;">
The property is a CSS property. The value is a CSS value.You will learn more about CSS later.

  • Use background-color property for background color.
  • Use color property for text colors.
  • Use font-family property for text fonts.
  • Use font-size property for text sizes.
  • Use text-align property for text alignment.
  • Use border property to define a borders.
  • Use border-collapse property to collapse cell borders.
  • Use padding property to add padding to cells.
  • Use border-spacing property to set the spacing between cells.
  • Use calspan attribute to make a cell span may columns.
  • Use rowspan attribute to make a cell span may rows.
  • Use id attribute to uniquely define one table.
